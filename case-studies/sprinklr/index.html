The Core Challenges: Why Redesign Sprinklr Community?

The existing dashboard was cluttered and lacked intuitive navigation, making it difficult for users to manage multiple social media accounts efficiently. The challenge was to create a clean, organized interface that would streamline the workflow while maintaining all essential functionality.

Low Engagement
  • Low participation rate
  • Low user retention
Cluttered UI
  • High cognitive load
  • Hard to navigate
  • No hierarchy
Dated Experience
  • Lack of modern UX patterns
  • Lack of consistency

Previous Sprinklr Community Design

The previous design of the Sprinklr Community lacked visual hierarchy and clarity, leading to confusion among users. The previous clutter layout has resulted in very high cognitive load and this makes it unnecessarily hard for users to navigate, read and perceive the contents.

Research

I engaged researchers early and arranged daily standups with researchers and PMs to understand use cases across different brands/personas and to identify their pain points.

Understanding Users' Needs

Users expressed a need for a more streamlined experience, with easy access to key features and a more visually appealing interface. Feedback highlighted the importance of customization and personalization in the dashboard.

1. Customization

Enterprise brands wanted customization without disrupting UX consistency.

2. Navigation

Users struggled to navigate cluttered layouts and find relevant discussions.

3. Gamification

Gamification elements were missing, leading to low user participation.

4. AI-Search

Integration of AI-powered features to boost user engagement.

How Might We Make Sprinklr Community More Engaging, Relevant, and Modern

By implementing a modular design approach, we can allow users to customize their dashboards with widgets that display relevant information. Additionally, enhancing the visual design with modern aesthetics will make the platform more engaging and user-friendly.

Modular UI
  • Applied Hyperspace Design System
  • Modular UI framework.
  • Consistent navigation patterns.
  • Content hierarchy improvements.
Gamification
  • Introduced a badge system for active contributors.
  • Added leaderboards showcasing top community members.
  • Rewarded users for their contributions.
AI-Search
  • Introduced AI search result explanations
  • Enhanced search ranking based on relevance, behavior, and engagement metrics

The Solution

Implemented a clean, modular dashboard design with improved navigation and visual hierarchy. Key improvements included:

  • Streamlined navigation with clear categorization
  • Customizable dashboard widgets
  • Improved data visualization
  • Enhanced mobile responsiveness

The Hyperspace Atomic Design System was developed to create a modern and consistent user interface that enhances user engagement across the Sprinklr platform.

  • Ensuring a cohesive & scalable UX across all brands.
  • Reusable patterns that make interactions more intuitive.
  • Making information scannable & easy to find.

Trade-Offs

While the new design improved user experience, it required trade-offs in terms of development time and resource allocation, which were carefully managed throughout the project.

  • More customization meant more complexity in the backend.
  • Gamification increased engagement, but some users felt competitive elements were unnecessary.

AI-Powered Search for Better Discovery & Engagement

The integration of AI-powered search capabilities enhanced content discovery and user engagement, allowing users to find relevant information quickly and efficiently.

The Trust Gap in AI-Powered Search & How Might We Fill in That Gap?

Addressing the trust gap in AI-powered search involved ensuring transparency in how search results are generated and providing users with control over their search experience.

Problem = Trust Issues

Users did not trust AI-generated search results due to lack of transparency.

Problem = Low Engagement

Users preferred manual browsing over AI-assisted suggestions.

Solution = AI-Explainability

Added "Why this result?" insights next to search results.

Solution = Blended Results

AI generated results mixed with traditional search.

AI Explainability & Blended Results

Ensuring AI explainability was crucial for user trust, and blended results provided a mix of AI-generated and user-generated content to enhance relevance.

Impact: Increased AI Adoption, Engagement & Reduced User Complaints

The implementation of AI features led to increased user adoption and engagement, while also reducing complaints related to search functionality.

35% AI Adoption Increase
20% User Engagement Increase
20% User Complaints Reduction

Trade-Offs

Balancing the benefits of AI features with user privacy and data security was a key consideration throughout the project.

Transparency

Making AI transparent reduced friction, but required more educational UI elements.

Discovery

Encouraging user discovery means more content will be introduced and required balancing to avoid overwhelming users.

What I Learned
